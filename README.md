# Basics of Git

This project is a simple Python example for practicing basic Git workflows.

## What this project covers
- A small Python script in main.py
- Basic repository setup and version control practice
- Simple documentation and project structure

## Getting started
1. Create and activate a virtual environment.
2. Install dependencies with uv:
   ```bash
   uv sync
   ```
3. Run the script:
   ```bash
   python main.py
   ```

## Project files
- main.py: contains the sample Python script
- pyproject.toml: project metadata and dependencies
- README.md: project overview and instructions
